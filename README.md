Notes website

A simple, personal notes website for organizing thoughts, code snippets, and study materials with a clean UI. Bulit for quick access and easy editing, ideal for students or developers tracking projects.

Features

	• Responsive design for desktop and mobile.

	• Markdown support for rich notes.

	• Local storage to save notes without a backend.

	• Search functionality for quick note finding.


Usage

	• Click "New Note" to start writing.

	• Format with Markdown ( eg. Heading/Title, -discription).

	• Search notes via the top bar.
